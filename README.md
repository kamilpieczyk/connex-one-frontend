* typescript "^4.5.5",
* web-vitals "^2.1.4"

# How to run the project

first install all required dependencies using command

```shell
npm install
```

or if you use yarn

```shell
yarn install
```

then run with command

```shell
npm run start
```

or

```shell
yarn start
```
